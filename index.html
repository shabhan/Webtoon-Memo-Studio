소개

웹툰 작가·편집자가 원고 이미지에 컷별 메모를 달고, 드로잉을 추가해 결과물을 PDF·JPG로 내보낼 수 있는 도구입니다. 작업 내용은 브라우저에 자동으로 저장됩니다.

기본 흐름
1
📁 새 작품 : 작품 폴더 만들기 상단 툴바의 새 작품 버튼을 클릭해 작품 제목을 입력합니다.
2
＋ 새 회차 : 원고 이미지 불러오기 새 회차 버튼을 클릭해 회차 번호를 입력하고, 원고 이미지(JPG/PNG)를 선택합니다.
파일명을 001.jpg, 002.jpg처럼 숫자 순서로 지정하면 원고가 올바른 순서로 배치됩니다.
3
🎨 메모·드로잉 추가 뷰어에 원고가 표시되면 이미지 위 원하는 위치를 클릭해 메모 카드를 추가합니다.
상단의 그리기 버튼으로 원고 위에 드로잉을 할 수 있습니다.
4
⬇ 내보내기 : PDF 또는 JPG 저장 작업이 끝나면 우측 상단 내보내기 버튼으로 결과물을 저장합니다.
일괄 불러오기

여러 회차 이미지를 한 번에 가져올 때 사용합니다. 파일명 패턴(예: {회차}화_{페이지}.jpg)을 지정하면 회차를 자동으로 분류합니다.

회차 내 페이지 추가 · 삭제
페이지 추가사이드바에서 회차의 ✎ 버튼 → 이미지 추가.
파일명 순으로 자동 정렬되며, 메모·드로잉 위치도 재매핑됩니다.
페이지 삭제원고 이미지에 마우스를 올리면 좌상단에 🗑 버튼이 나타납니다.
클릭하면 해당 페이지와 그 페이지의 메모가 삭제됩니다.
삭제된 페이지와 메모는 복원할 수 없습니다. 삭제 전 백업을 권장합니다.
카드 편집
드래그카드 위치 이동. 다른 페이지로 이동하면 페이지 번호가 자동 변경됩니다.

분할 수: 웹툰 원고는 세로가 길어 한 이미지를 여러 PDF 페이지로 나누어 저장합니다. 툴바의 − / ＋ 버튼으로 분할 수를 조절합니다.

1~2만px2분할 권장

JPG 내보내기

메모가 있는 페이지는 오른쪽에 메모 컬럼이 추가되고, 메모가 없는 페이지는 원본 해상도 그대로 저장됩니다.

PDF 내보내기는 최초 1회 인터넷에서 라이브러리를 다운로드합니다. 오프라인에서는 JPG 내보내기를 사용하세요.
자동저장

작업 내용은 브라우저(IndexedDB)에 자동 저장됩니다. 우측 하단에 자동저장됨 표시가 나타나면 저장 완료! 브라우저를 닫았다 다시 열어도 복원됩니다.

자동저장은 이 브라우저에만 저장됩니다. 다른 기기·다른 브라우저에서는 백업 불러오기를 사용하세요.
백업 저장 / 불러오기
💾 백업 저장전체 작업(이미지 포함)을 JSON 파일로 저장합니다. 정기적으로 백업하는 것을 권장합니다.
🔃 백업 불러오기저장된 JSON을 불러와 복원합니다. 현재 작업이 교체되므로, 불러오기 전 백업을 먼저 저장하세요.
브라우저 캐시를 지우면 자동저장 데이터가 삭제될 수 있습니다. 중요한 작업은 반드시 백업 저장을 해주세요.
